# Student Score Prediction App  
A web app built with Streamlit for predicting student scores based on study hours and other features.

## Live Demo  
Access the live app here:  
[Student Score Prediction](https://finalstudentscorebypria.streamlit.app/)

##  Purpose  
This app helps students, educators, or data-science enthusiasts to:  
- Estimate the expected exam score based on input features like weekly study hours.  
- Understand relationships between study habits and performance.  
- Explore a simple machine learning use-case end-to-end.

##  Tech Stack  
- **Frontend & deployment**: [Streamlit](https://streamlit.io)  
- **Backend & Machine Learning**: Python (libraries such as scikit-learn, pandas, numpy)  
- **Model**: A regression model trained to predict student scores  
- **Hosting**: Streamlit sharing / Streamlit cloud  

##  Features  
- Slider input for "Study Hours per Week" (range 0-40, default 10)  
- Real-time result display of predicted student score  
- Simple, intuitive UI for quick experimentation  
- Clean codebase for educational and demonstration purposes
